# CosmicDan's MIUI Kitchen



## Required binaries/packages:

* brotli
* simg2img

E.g:

`sudo apt install brotli android-tools-fsutils`



## Usage

Run `. ./build/envsetup.sh` (just like REAL Android!) then use `help` command.
